# react-bejeweled

A Project based on the Games Bejeweled and Candy Crush, created with the Javascript library React.js

Functionality:
- Create 8x8 board with random elements of 7 different colors
- Further randomize elements before initializing board
- Monitor board for possible moves - Game Over if no more moves exist
- Functions to detect 3, 4 and 5 elements of the same color in a column and row
- Functions for Drag & Drop functionality
- Function to refill board with elements once matches have been removed
